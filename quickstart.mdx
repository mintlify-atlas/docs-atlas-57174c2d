## Get your first widget running

This guide will walk you through adding the Hydration widget to your Corsair iCUE setup. The same process works for any iframe-edge widget.

<Steps>
  <Step title="Open iCUE">
    Launch the Corsair iCUE software and navigate to your Xeneon Edge display configuration.
  </Step>

  <Step title="Add an iframe widget">
    In the **Widgets** list on the left sidebar, click the **`</> iFrame`** icon.
    
    Choose your desired size:
    - **M** - Compact layout for smaller displays
    - **L** - Standard layout with more breathing room
    - **XL** - Full-featured layout with maximum space
  </Step>

  <Step title="Paste the iframe code">
    In the iframe settings panel, locate the **HTML code** text area and paste this code:

    ```html
    <iframe src="https://stealthylabshq.github.io/iframe-edge/productivity/hydration/"></iframe>
    ```

    <Tip>
      The widget loads directly from GitHub Pages, so you'll always get the latest version automatically.
    </Tip>
  </Step>

  <Step title="See it in action">
    The Hydration widget will load immediately on your Xeneon Edge display. You'll see:
    - A water glass icon and title
    - Your daily progress (0 / 8 glasses)
    - Empty glass indicators
    - A **+1 glass** button to track your hydration
  </Step>
</Steps>

## Understanding the widget

The Hydration widget you just added is a fully self-contained application. Here's what makes it work:

### HTML structure

The widget uses semantic HTML with a clean component hierarchy:

```html
<div class="widget">
  <div class="mod-header">
    <div class="mod-icon">
      <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2.5"
        stroke-linecap="round">
        <path d="M12 2.69l5.66 5.66a8 8 0 1 1-11.31 0z" />
      </svg>
    </div>
    <span class="mod-title" id="t-title">Hydration</span>
    <div class="lang-toggle" id="themeToggle" style="margin-left: auto">
      🌙
    </div>
    <div class="lang-toggle" id="langToggle">EN</div>
  </div>
  <div class="water-body">
    <div class="water-count-row">
      <div class="water-number" id="waterNum">0</div>
      <div class="water-meta" id="t-meta">/ 8 glasses · 2 L</div>
    </div>
    <div class="glass-grid" id="glassGrid"></div>
    <div class="water-status" id="waterStatus">Start hydrating!</div>
    <div class="water-controls">
      <button class="btn btn-ghost" id="waterMinus" aria-label="Remove a glass">
        <!-- minus icon -->
      </button>
      <button class="btn btn-water" id="waterPlus" aria-label="Add a glass">
        <!-- water droplet icon -->
        <span id="t-add">+1 glass</span>
      </button>
    </div>
  </div>
</div>
```

### JavaScript architecture

The widget's logic is pure vanilla JavaScript with no dependencies:

```javascript
const $ = (id) => document.getElementById(id);

function showToast(msg, duration = 3000) {
  const el = $("toast");
  el.textContent = msg;
  el.classList.add("visible");
  clearTimeout(el._t);
  el._t = setTimeout(() => el.classList.remove("visible"), duration);
}

function today() {
  return new Date().toISOString().split("T")[0];
}

const THEME_KEY = "pa_theme";
let currentTheme = localStorage.getItem(THEME_KEY) || "dark";

function updateThemeUI() {
  $("themeToggle").textContent = currentTheme === "dark" ? "🌙" : "☀️";
  document.documentElement.setAttribute("data-theme", currentTheme);
}

$("themeToggle").addEventListener("click", () => {
  currentTheme = currentTheme === "dark" ? "light" : "dark";
  localStorage.setItem(THEME_KEY, currentTheme);
  updateThemeUI();
});
```

<Note>
  The widget uses localStorage to persist your theme preference and daily water count. This data is stored locally in your browser and never sent to any server.
</Note>

## Try the widget features

Now that your widget is running, explore these features:

1. **Track your hydration** - Click the **+1 glass** button to log water intake
2. **Toggle theme** - Click the moon icon (🌙) to switch between dark and light modes
3. **Change language** - Click **EN** to toggle between English and French
4. **Reset progress** - Use the minus button to adjust your count if needed

<Info>
  Your hydration count automatically resets at midnight each day. The widget shows encouraging messages as you progress toward your daily goal of 8 glasses.
</Info>
