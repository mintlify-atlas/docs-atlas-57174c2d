## What is Corsair iCUE?

Corsair iCUE (Intelligent Corsair Utility Engine) is Corsair's powerful software platform that enables you to control and customize your Corsair devices. With iCUE, you can manage RGB lighting, monitor system performance, and configure widgets on compatible displays like the Corsair Xeneon Edge.

For iframe-edge widgets, iCUE serves as the platform that hosts and displays web-based widgets directly on your Xeneon Edge display through its built-in iFrame widget functionality.

## Prerequisites

Before installing iCUE, ensure your system meets the following requirements:

- **Display**: Corsair Xeneon Edge monitor
- **Operating System**: Windows 10/11 or macOS
- **Internet Connection**: Required for downloading software and loading widgets
- **Disk Space**: Approximately 500 MB for iCUE installation

<Note>
The Corsair Xeneon Edge display is required to use the dashboard and widget features. Other Corsair monitors may not support the iFrame widget functionality.
</Note>

## Installation Steps

<Steps>
  <Step title="Download iCUE">
    Visit the official Corsair iCUE download page:
    
    [https://www.corsair.com/icue](https://www.corsair.com/icue)
    
    Click the download button to get the latest version of iCUE for your operating system.
  </Step>

  <Step title="Run the Installer">
    Once the download completes, run the installer executable:
    - **Windows**: Double-click the `.exe` file
    - **macOS**: Open the `.dmg` file and follow the installation prompts
    
    Grant administrator permissions when prompted.
  </Step>

  <Step title="Complete Installation">
    Follow the on-screen installation wizard:
    1. Accept the license agreement
    2. Choose your installation directory (or use the default)
    3. Wait for the installation to complete
    4. Launch iCUE when installation finishes
  </Step>

  <Step title="Connect Your Devices">
    Ensure your Corsair Xeneon Edge display is connected to your computer via:
    - DisplayPort or HDMI (for video)
    - USB cable (for iCUE communication)
    
    iCUE should automatically detect your connected Corsair devices.
  </Step>
</Steps>

<Warning>
Make sure to keep iCUE up to date to ensure compatibility with the latest widgets and features. iCUE will notify you when updates are available.
</Warning>

## Initial Configuration

After installing iCUE:

1. **Launch iCUE** from your applications menu or desktop shortcut
2. **Check for Updates**: Click on the settings icon and check for software updates
3. **Device Detection**: Verify that your Xeneon Edge display appears in the device list
4. **Firmware Updates**: If prompted, update your device firmware to the latest version

<Info>
iCUE may require a restart after initial installation or firmware updates. Make sure to save any open work before proceeding.
</Info>

## Accessing the Dashboard

Once iCUE is installed and configured:

<Steps>
  <Step title="Open iCUE">
    Launch the iCUE application from your desktop or applications folder.
  </Step>

  <Step title="Select Your Display">
    In the iCUE interface, locate your Corsair Xeneon Edge display in the device list on the home screen.
    
    Click on the display to access its configuration options.
  </Step>

  <Step title="Access Screen Configuration">
    Navigate to your screen/dashboard configuration section. This is where you can:
    - Add and remove widgets
    - Customize widget layouts
    - Configure dashboard settings
  </Step>

  <Step title="View Available Widgets">
    On the left side of the configuration screen, you'll see the **Widgets** list. This includes:
    - Built-in Corsair widgets (clock, system monitoring, etc.)
    - The **`</> iFrame`** widget for custom web content
  </Step>
</Steps>

<Note>
The dashboard configuration interface shows a live preview of how your widgets will appear on the Xeneon Edge display.
</Note>
