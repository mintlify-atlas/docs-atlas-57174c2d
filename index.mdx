## Key features

iframe-edge widgets are designed to be plug-and-play while remaining powerful and flexible:

- **10+ self-contained productivity widgets** - Track hydration, manage budgets, run Pomodoro timers, take notes, and more
- **AI assistant with 3 providers** - Choose between Google Gemini, Anthropic Claude, and OpenAI ChatGPT
- **Spotify visualizer** - Real-time music visualization with lyrics for Spotify Premium users
- **No external dependencies** - Pure HTML, CSS, and vanilla JavaScript bundled in single files
- **Fully responsive** - Automatically scales to fit iCUE container sizes (M, L, XL)
- **Dark and light mode** - Toggle theme on every widget with persistent preferences
- **Bilingual support** - Switch between English and French on all widgets
- **Local storage persistence** - Your settings and data stay with you

## Architecture overview

Each widget follows a consistent, minimal architecture:

<Steps>
  <Step title="Single-file design">
    Every widget is contained in one HTML file that includes inline CSS and JavaScript. No bundlers, no transpilers, no package managers.
  </Step>
  <Step title="Vanilla JavaScript">
    Pure JavaScript with no frameworks or libraries. The code is readable, maintainable, and easy to customize.
  </Step>
  <Step title="Theme system">
    A shared `theme-loader.js` script manages dark/light mode preferences using CSS custom properties and localStorage.
  </Step>
  <Step title="Internationalization">
    Built-in i18n with English and French translations, implemented as simple JavaScript objects.
  </Step>
</Steps>

<Info>
  All widgets use localStorage to persist user settings and data. This means your preferences, theme choices, and widget state are preserved across sessions.
</Info>

## Use cases

iframe-edge widgets are perfect for:

- **Streaming setups** - Display productivity tools or music visualizers on your Xeneon Edge while streaming
- **Productivity dashboards** - Keep track of hydration, tasks, and time management without leaving your main workflow
- **Development environments** - Quick access to notes, clipboard history, and timers while coding
- **Content creation** - Monitor habits, focus sessions, and break reminders during long creative sessions
- **AI-assisted workflows** - Instant access to AI chat without switching windows or contexts

## What's inside

The project includes:

- **Productivity widgets**: Hydration tracker, budget manager, Pomodoro timer, quick notes, daily focus, habit tracker, posture reminders, quick clipboard, and timer
- **AI & streaming**: AI assistant (Gemini/Claude/ChatGPT), Spotify visualizer with lyrics, news radar
- **Theme support**: Dark and light modes with smooth transitions
- **Localization**: English and French language support
