The Quick Clipboard widget provides a persistent textarea for storing text snippets, links, scripts, or commands that you need frequent access to. It includes one-click copy functionality and auto-save.

## Key Features

- **One-click copy**: Copy entire content to clipboard with single button
- **Auto-save**: Content saves 300ms after you stop typing
- **Persistent storage**: Retains content across sessions
- **Copy fallback**: Uses `document.execCommand` for iframe compatibility
- **Clear button**: Quick delete with visual confirmation
- **Bilingual**: French/English support
- **No spell check**: Clean typing for code and commands

## Copy Mechanism

The widget uses a fallback copy method that works in iframes:

```javascript
$("copyBtn").addEventListener("click", () => {
  const text = clipArea.value;
  if (!text.trim()) {
    showToast(t("toastEmpty"));
    return;
  }

  // Create a temporary off-screen textarea
  const tmp = document.createElement("textarea");
  tmp.value = text;
  tmp.style.position = "fixed";
  tmp.style.top = "-9999px";
  tmp.style.left = "-9999px";
  tmp.setAttribute("readonly", "");
  document.body.appendChild(tmp);
  tmp.focus();
  tmp.select();
  tmp.setSelectionRange(0, 99999);
  
  try {
    const ok = document.execCommand("copy");
    showToast(ok ? t("toastCopied") : "❌ Copy failed.");
  } catch (e) {
    showToast("❌ Copy failed.");
  }
  
  document.body.removeChild(tmp);
});
```

<Info>
This widget uses `document.execCommand('copy')` instead of the modern `navigator.clipboard` API because it needs to work inside iframes. The modern API often fails in embedded contexts due to security restrictions.
</Info>

## Auto-Save Implementation

```javascript
const CLIP_KEY = "pa_quickClipboard";
let saveTimer = null;
const clipArea = $("clipArea");

clipArea.value = localStorage.getItem(CLIP_KEY) || "";

clipArea.addEventListener("input", () => {
  clearTimeout(saveTimer);
  saveTimer = setTimeout(() => {
    localStorage.setItem(CLIP_KEY, clipArea.value);
  }, 300);
});
```

The 300ms debounce is shorter than Notes (500ms) because clipboard content is often copied quickly - you don't want to lose snippets due to slow saves.

## Clear Functionality

```javascript
$("clearBtn").addEventListener("click", () => {
  clipArea.value = "";
  localStorage.removeItem(CLIP_KEY);
  showToast(t("toastCleared"));
});
```

<Note>
Unlike the Notes widget, the Clear button does NOT ask for confirmation. This is intentional - clipboard content is usually temporary, and accidental clears are less catastrophic.
</Note>

## Toast Notifications

The widget shows toast messages for:

- **Copy success**: "✅ Copied to clipboard!"
- **Copy failure**: "❌ Copy failed."
- **Empty copy attempt**: "⚠️ Nothing to copy."
- **Clear action**: "🗑️ Clipboard cleared."

## Differences from Notes Widget

| Feature | Quick Clipboard | Notes |
|---------|----------------|-------|
| **Purpose** | Temporary storage, copy | Permanent note-taking |
| **Auto-save delay** | 300ms | 500ms |
| **Clear confirmation** | No | Yes |
| **Character counter** | No | Yes |
| **Save badge** | No | Yes |
| **Main action** | Copy button | None (just saves) |

<Tip>
Use Quick Clipboard for things you copy frequently: commands, URLs, code snippets, email templates. Use Notes for longer-form thoughts and ideas.
</Tip>

## Security Considerations

<Info>
**Do NOT store sensitive data in this widget:**
- Passwords or API keys
- Personal identification numbers
- Credit card information
- Private tokens

All content is stored in localStorage, which is accessible to any JavaScript on the same domain.
</Info>

## Iframe Compatibility

The copy mechanism is specifically designed for iframe contexts:

```javascript
// Modern API (doesn't work in many iframes)
navigator.clipboard.writeText(text); // ❌

// Fallback method (works in iframes)
document.execCommand('copy'); // ✅
```

<Info>
The widget creates a temporary textarea, selects its content, and executes the copy command. This bypasses the security restrictions on `navigator.clipboard` in embedded contexts.
</Info>
