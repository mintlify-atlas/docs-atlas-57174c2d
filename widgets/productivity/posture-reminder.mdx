The Posture Reminder widget implements the 20-20-20 rule to protect your eyes and remind you to maintain good posture. Every 20 minutes, it prompts you to look 20 feet away for 20 seconds and check your posture.

## Key Features

- **20-minute timer**: Countdown to next reminder
- **Visual alert**: Widget changes appearance when time's up
- **Health reminders**: Posture, eye rest, and blinking prompts
- **Auto-start**: Begins counting immediately on load
- **One-click reset**: Acknowledge and restart with single button
- **Bilingual**: French/English toggle
- **Always running**: Continuous loop for all-day protection

## The 20-20-20 Rule

The widget implements a scientifically-backed health practice:

<Info>
**Every 20 minutes:**
- Look at something 20 feet (6 meters) away
- For at least 20 seconds
- This reduces eye strain and digital fatigue
</Info>

The widget adds posture and blinking reminders:
- **Sit up straight**: Check your back and shoulder alignment
- **Blink your eyes**: Combat dry eyes from reduced blink rate

## Timer Implementation

```javascript
const INTERVAL_S = 20 * 60; // 20 minutes
let timeLeft = INTERVAL_S;
let isAlert = false;
let timer = null;

timer = setInterval(() => {
  if (!isAlert) {
    timeLeft--;
    renderTimer();
    if (timeLeft <= 0) {
      triggerAlert();
    }
  }
}, 1000);
```

## States

### Working State

- **Display**: Countdown timer showing remaining time
- **Message**: "Focusing. Next friendly reminder in 20m."
- **Button**: "Reset" (allows manual restart)
- **Visual**: Normal widget appearance

### Alert State

- **Display**: "00:00" (time expired)
- **Message**: "Sit up straight. Look 20ft away for 20s. Blink your eyes."
- **Button**: "Done! Resume" 
- **Visual**: Widget adds `alert-mode` class for styling

## Alert Trigger

```javascript
function triggerAlert() {
  isAlert = true;
  $("widget").classList.add("alert-mode");
  updateLangUI();
}
```

## Reset Functionality

```javascript
function resetTimer() {
  isAlert = false;
  timeLeft = INTERVAL_S;
  $("widget").classList.remove("alert-mode");
  updateLangUI();
  renderTimer();
}

$("actionBtn").addEventListener("click", () => {
  resetTimer();
});
```

## Time Display

```javascript
function formatTime(s) {
  if (isAlert) return "00:00";
  return `${String(Math.floor(s / 60)).padStart(2, "0")}:${String(s % 60).padStart(2, "0")}`;
}
```

## Health Benefits

### Eye Strain Prevention

<Info>
**Digital Eye Strain (Computer Vision Syndrome) symptoms:**
- Dry eyes
- Blurred vision  
- Headaches
- Neck and shoulder pain

The 20-20-20 rule helps prevent these symptoms.
</Info>

### Posture Benefits

- **Reduced back pain**: Regular posture checks prevent slouching
- **Better breathing**: Upright posture opens chest cavity
- **Increased energy**: Good posture improves circulation
- **Less neck strain**: Proper head position reduces tension

## Manual Reset

You can manually reset the timer at any time:

- **During countdown**: Restarts from 20:00 (useful if you just took a break)
- **During alert**: Dismisses alert and starts new cycle

## Customization Ideas

While the widget is hardcoded to 20 minutes, you could modify `INTERVAL_S`:

```javascript
// Default: 20 minutes
const INTERVAL_S = 20 * 60;

// Alternative: 25 minutes (Pomodoro)
const INTERVAL_S = 25 * 60;

// Alternative: 30 minutes
const INTERVAL_S = 30 * 60;
```

<Note>
However, 20 minutes is recommended by eye care professionals. Going longer increases eye strain risk.
</Note>

## Best Practices

<Tip>
**When the alert appears:**
1. Stand up if possible
2. Roll your shoulders back
3. Sit/stand with aligned spine
4. Look at something far away (out a window is ideal)
5. Blink deliberately 10-20 times
6. Take 2-3 deep breaths
7. Click "Done! Resume" and continue working
</Tip>
