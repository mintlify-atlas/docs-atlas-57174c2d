The Pomodoro widget implements the classic Pomodoro Technique with 25-minute focus sessions and 5-minute breaks. It features a beautiful animated progress ring, session tracking, and automatic phase transitions.

## Key Features

- **Classic Pomodoro timing**: 25 minutes focus, 5 minutes break
- **Session dots**: Visual tracker for 4-pomodoro cycles
- **Animated progress ring**: SVG circle with smooth animation
- **Phase badges**: Color-coded focus/break indicators
- **Auto-phase transition**: Automatically switches between work and break
- **Skip phase**: Jump to next phase without waiting
- **Bilingual**: French/English toggle
- **Running state visual**: Widget changes appearance when active

## Timer Phases

### Focus Phase (25 minutes)

- Badge: Red/Orange "Focus" label
- Ring color: Primary accent color
- Sublabel: "focus"
- On completion: Increments session counter, switches to break

### Break Phase (5 minutes)

- Badge: Blue "Break" label  
- Ring color: Break-specific color (teal/blue)
- Sublabel: "break" or "pause"
- On completion: Returns to focus phase

## Progress Ring

The widget uses SVG circles to create a smooth countdown ring:

```javascript
const FOCUS_S = 25 * 60;
const BREAK_S = 5 * 60;
const CIRC = 2 * Math.PI * 42; /* 263.89 */

function renderTimer() {
  timerDigits.textContent = formatTime(timer.timeLeft);
  const offset = CIRC * (1 - timer.timeLeft / timer.totalTime);
  ringProgress.style.strokeDashoffset = offset;
}
```

The ring uses `stroke-dasharray` and `stroke-dashoffset` to animate:

```html
<circle class="ring-progress" 
  cx="50" cy="50" r="42" 
  stroke-dasharray="263.89"
  stroke-dashoffset="0" />
```

## Session Tracking

The widget tracks up to 4 pomodoros per cycle:

```javascript
function renderSessionDots() {
  document.querySelectorAll(".s-dot").forEach((dot, i) => {
    dot.classList.toggle("done", i < timer.sessions);
  });
}
```

When all 4 sessions are complete, the counter resets and shows "🎉 Cycle complete!"

## Controls

### Start/Pause Button

- **Start**: Begin countdown from current time
- **Pause**: Stop the timer without resetting
- **Resume**: Continue from paused time

The button icon changes between play ▶ and pause ⏸:

```javascript
const ICON_PLAY = "M5 3 19 12 5 21 5 3";
const ICON_PAUSE = "M6 4h4v16H6zM14 4h4v16h-4z";

function setRunningUI(running) {
  if (running) {
    startLabel.textContent = t("btnPause");
    startIcon.setAttribute("d", ICON_PAUSE);
    widget.classList.add("is-running");
  } else {
    startLabel.textContent = 
      timer.timeLeft < timer.totalTime ? t("btnResume") : t("btnStart");
    startIcon.setAttribute("d", ICON_PLAY);
  }
}
```

### Reset Button

Resets the current phase to its full duration without changing the phase type.

### Skip Button

Skips to the next phase immediately, triggering the phase end logic.

## Phase Transitions

```javascript
function handlePhaseEnd() {
  clearInterval(timer.interval);
  timer.interval = null;
  timer.running = false;
  
  if (!timer.isBreak) {
    timer.sessions = Math.min(timer.sessions + 1, 4);
    renderSessionDots();
    timer.isBreak = true;
    timer.totalTime = BREAK_S;
    timer.timeLeft = BREAK_S;
    renderPhase();
    showToast(t("toastFocusEnd"));
  } else {
    timer.isBreak = false;
    timer.totalTime = FOCUS_S;
    timer.timeLeft = FOCUS_S;
    if (timer.sessions >= 4) {
      timer.sessions = 0;
      renderSessionDots();
      showToast(t("toastCycleEnd"));
    } else {
      showToast(t("toastNextFocus"));
    }
    renderPhase();
  }
}
```

## Time Formatting

The timer displays time in MM:SS format:

```javascript
function formatTime(s) {
  return `${String(Math.floor(s / 60)).padStart(2, "0")}:${String(s % 60).padStart(2, "0")}`;
}
```

## Toast Notifications

The widget shows toast messages for:

- "🍅 Focus complete! 5 min break." - After focus session
- "💪 Here we go again! New focus." - Starting new focus after break
- "🎉 Cycle complete! 4 pomodoros finished." - After completing 4 sessions

<Info>
The Pomodoro Technique was developed by Francesco Cirillo in the late 1980s. The technique uses a timer to break work into intervals, traditionally 25 minutes in length, separated by short breaks.
</Info>

<Tip>
For best results, use the Pomodoro timer for focused deep work. During the 25-minute focus sessions, eliminate all distractions. During the 5-minute breaks, step away from your desk to rest your eyes and mind.
</Tip>
