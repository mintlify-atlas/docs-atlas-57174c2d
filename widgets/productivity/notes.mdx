The Quick Notes widget provides a lightweight, auto-saving textarea for capturing ideas, tasks, and quick thoughts. It features character counting, visual save indicators, and instant persistence.

## Key Features

- **Auto-save**: Saves automatically 500ms after you stop typing
- **Character counter**: Live count of characters written
- **Save indicator**: Visual checkmark when content is saved
- **Clear button**: Quickly delete all notes with confirmation
- **No limit**: Unlimited text storage (localStorage limits apply)
- **Spell check disabled**: Clean typing experience without distractions
- **Bilingual**: French/English support
- **Instant load**: Notes restore immediately on page load

## Auto-Save Mechanism

The widget uses a debounced save pattern:

```javascript
const NOTES_KEY = "pa_quickNotes";
let saveTimer = null;
const notesArea = $("notesArea");

notesArea.addEventListener("input", () => {
  updateCharCount();
  clearTimeout(saveTimer);
  saveTimer = setTimeout(() => {
    localStorage.setItem(NOTES_KEY, notesArea.value);
    flashSaved();
  }, 500);
});
```

<Info>
The 500ms delay ensures the widget doesn't save on every keystroke. It waits until you pause typing, reducing localStorage writes and improving performance.
</Info>

## Visual Save Feedback

When content is saved, a checkmark badge appears:

```javascript
function flashSaved() {
  saveBadge.classList.add("show");
  clearTimeout(saveBadge._t);
  saveBadge._t = setTimeout(
    () => saveBadge.classList.remove("show"),
    2200,
  );
}
```

The badge includes an SVG checkmark icon:

```html
<span class="save-badge" id="saveBadge">
  <svg width="11" height="11" viewBox="0 0 24 24" 
    stroke="currentColor" stroke-width="3" stroke-linecap="round">
    <polyline points="20 6 9 17 4 12" />
  </svg>
  <span id="t-saved">Saved</span>
</span>
```

## Character Counter

The widget displays a real-time character count:

```javascript
function updateCharCount() {
  const n = notesArea.value.length;
  charCount.textContent = t("char", n);
}
```

With bilingual support:

```javascript
const i18n = {
  fr: {
    char: (n) => `${n} caractère${n !== 1 ? "s" : ""}`,
  },
  en: {
    char: (n) => `${n} character${n !== 1 ? "s" : ""}`,
  },
};
```

## Clear Functionality

The clear button includes a confirmation dialog:

```javascript
$("clearBtn").addEventListener("click", () => {
  if (!notesArea.value) return;
  if (window.confirm(t("confirm"))) {
    notesArea.value = "";
    localStorage.removeItem(NOTES_KEY);
    updateCharCount();
    showToast(t("toastClear"));
  }
});
```

<Note>
The clear button is non-destructive - it always asks for confirmation before deleting your notes.
</Note>

## Local Storage Keys

| Key | Purpose | Type |
|-----|---------|------|
| `pa_quickNotes` | Note content | String |
| `pa_theme` | Theme preference | "dark" or "light" |
| `pa_lang` | Language preference | "fr" or "en" |

## Load on Startup

Notes restore automatically when the widget loads:

```javascript
function loadNotes() {
  notesArea.value = localStorage.getItem(NOTES_KEY) || "";
  updateCharCount();
}

loadNotes();
```

## HTML Structure

The textarea is configured for optimal note-taking:

```html
<textarea 
  class="notes-textarea" 
  id="notesArea" 
  placeholder="Your ideas, tasks, thoughts... Auto-saving."
  spellcheck="false"></textarea>
```

`spellcheck="false"` prevents distracting red underlines while typing quick notes.

## Use Cases

- **Quick thoughts**: Jot down ideas as they come
- **Temporary clipboard**: Store text snippets between apps
- **Meeting notes**: Take quick notes during calls
- **To-do lists**: Write simple task lists
- **Code snippets**: Store commands or code temporarily
- **URLs**: Keep links for later reference

<Tip>
Use this widget as a scratch pad for anything you need to remember temporarily. Since it auto-saves, you won't lose your thoughts even if you close the browser.
</Tip>

<Info>
While there's no hard character limit in the code, localStorage typically has a limit of 5-10MB per domain. For very long documents, consider using a dedicated note-taking app.
</Info>
