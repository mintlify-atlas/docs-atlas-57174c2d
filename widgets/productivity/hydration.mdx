The Hydration widget helps you track your daily water consumption with a goal of 8 glasses (2 liters) per day. It features a visual glass grid, automatic daily reset, and bilingual support.

## Key Features

- **Daily goal tracking**: Track progress toward 8 glasses of water (2L)
- **Visual glass grid**: See filled glasses (💧) vs empty circles (○)
- **Auto-save progress**: Saves to localStorage automatically
- **Daily reset**: Automatically resets counter at midnight
- **Progress messages**: Dynamic status showing glasses remaining
- **Bilingual**: French/English toggle
- **Theme support**: Light/dark mode with sync across tabs

## How It Works

### Water Tracking

The widget tracks your daily water intake with these states:

1. **Not started**: "Start hydrating!" message
2. **In progress**: "X glass(es) left for your goal"
3. **Goal reached**: "🎉 Daily goal reached!"

### Daily Reset Logic

The widget automatically checks if it's a new day:

```javascript
function today() {
  return new Date().toISOString().split("T")[0];
}

function loadWater() {
  const savedDate = localStorage.getItem(WATER_DATE_KEY);
  if (savedDate !== today()) {
    localStorage.setItem(WATER_DATE_KEY, today());
    localStorage.setItem(WATER_KEY, "0");
    return 0;
  }
  return parseInt(localStorage.getItem(WATER_KEY) || "0", 10);
}
```

Every 60 seconds, it checks if the date has changed:

```javascript
setInterval(() => {
  const saved = localStorage.getItem(WATER_DATE_KEY);
  if (saved && saved !== today()) {
    waterCount = 0;
    saveWater(0);
    renderWater();
    showToast(t("toastReset"));
  }
}, 60_000);
```

## Local Storage Keys

| Key | Purpose | Type |
|-----|---------|------|
| `pa_waterCount` | Current water count | Number (0-8) |
| `pa_waterDate` | Date of last water entry | ISO date string (YYYY-MM-DD) |
| `pa_theme` | Theme preference | "dark" or "light" |
| `pa_lang` | Language preference | "fr" or "en" |

## Visual Feedback

The glass grid provides instant visual feedback:

```javascript
function buildGlassGrid() {
  const grid = $("glassGrid");
  grid.replaceChildren();
  for (let i = 0; i < WATER_GOAL; i++) {
    const cell = document.createElement("div");
    cell.className = "glass-cell" + (i < waterCount ? " filled" : "");
    cell.textContent = i < waterCount ? "💧" : "○";
    cell.style.fontSize = i < waterCount ? "18px" : "16px";
    cell.style.color = i < waterCount ? "" : "var(--text-3)";
    grid.appendChild(cell);
  }
}
```

## Cross-Tab Sync

The widget listens for storage events to sync across browser tabs:

```javascript
window.addEventListener("storage", (e) => {
  if (e.key === LANG_KEY) {
    currentLang = e.newValue;
    updateLangUI();
  }
});
```
